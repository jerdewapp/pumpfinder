# 🚴 PumpFinder

Application mobile (PWA) de recensement des pumptracks en France.

## Fonctionnalités
- 🗺️ Carte interactive avec 1229 pumptracks recensés
- 📍 Géolocalisation et spot le plus proche
- 🔍 Recherche et filtres (difficulté, constructeur, surface…)
- 📋 Fiche détail avec équipements, signalement communautaire
- ➕ Ajout de spots par la communauté
- 🇫🇷 / 🇬🇧 Bilingue français / anglais

## Données
Spots issus de [ourouler.com](https://www.ourouler.com) — la référence française du pumptrack.

## Stack
- HTML / CSS / JS vanilla
- Leaflet.js + OpenStreetMap
- GitHub Pages
